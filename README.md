### how to
## development
1. clone the repo
2. install uv package manager:
\>\>\> python3 -m pip install -U pip && pip install uv
3. Build the project:
\>\>\> uv build
4. Install the package:
\>\>\> uv tool install --force dist/hexlet_code-0.1.0-py3-none-any.whl
5. Run the game:
\>\>\> brain-even
